    Evenbac

    Evenbac is a project that aims to bring to my local community event announcements that happen in my area, where all users can post their own events and details about it and other people can see it.     UBBNews

    I launched an app that helps students keep track of the announcements posted on a professor's page. During the online classes, following the COVID-19 pandemic lockdown measures, I found it quite difficult to keep track of every post on a professor's website, as there were so many. Therefore, I created UBBNews, an app that uses web scraping techniques to check the given urls for changes on the web page. It currently has over 100 downloads on Google Play
